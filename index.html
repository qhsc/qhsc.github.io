    1. 最后一块石头的重量 II
    • 题目：给定一堆石头，每个石头有重量 stones[i]。执行以下步骤直到只剩一个石头：任选两个石头进行碰撞相消。如果重量相等，则碰撞以后不会留下任何石头；如果重量不等， 两块石头碰撞以后会留下一块石头，重量为两块石头重量的差值。问：留下的石头的重量最小可能是多少？
    • 解析：
      • 分析：每次碰撞都会消去2的整数倍的重量，则最终被消去的重量是 2*x，这里 x 是正整数，还留下 y 重量的石头。则有 sum(stones)=2*x+y。因此，如果我们能把石头分作两堆，并且两堆的差距是最小的，则此时的差距就是 y ，也就是题目所求。
      • 解法：上述问题可以转化为：从一堆数字中选取若干个，其能得到的和当中不大于且最接近这一堆数的和的一半的数是多少。这是一个 0-1 背包问题。用动态规划求解。
    • 代码
      class Solution {
public:
    int lastStoneWeightII(vector<int>& stones) {
        int total = 0;
        for(int i=0; i<stones.size(); i++) {
            total += stones[i];
        }

        int half = total/2;
        vector<int> possibleMaxSum(half+1, 0);
        for (int index=0; index<stones.size(); index++) {
            int currentStone = stones[index];
            for (int sum=half; sum >= currentStone; sum--) {
                possibleMaxSum[sum] = max(possibleMaxSum[sum],
                    currentStone+possibleMaxSum[sum-currentStone]);
            }
        }
        return total - (possibleMaxSum.back() << 1);
    }
};
    1. 填充书架
    • 题目：你现在有一个书架宽度 width 是固定的，层数是可以拓展的，然后还有一堆书，这堆书已经放好了顺序，每本书都有自己的厚度 thickness_i 和 高度 height_i。你需要按照顺序将这些书放到书架里面，排放的方式就像图书馆的书一样，即：每一层的书籍都是竖起来的，它们的厚度加起来不能超过书架的宽度，然后每一层的宽度由这一层书当中最高的决定，一层书可以不放满。求最后的书架的最小高度是多少？
    • 解析：采用典型的动态规划算法，假设 miniHeight[i] 代表存放第 i 本书后书架的最低高度，当新加一本书时，如何推导第 i+1 本书所需的最低高度呢？新加的这一本书一定属于最后一层，则书架的高度一定等于上一层的高度加上本层的高度。我们假设层高为 height，则有 miniHeight[i+1]=miniHeight[x]+height && sum(thickness[x+1,i+2]) <= width && x >= -1 && x <= i。
    • 代码
      class Solution {
public:
    int minHeightShelves(vector<vector<int>>& books, int shelfWidth) {
        vector<int> miniHeight(books.size(), 0);
        miniHeight[0] = books[0][1];

        for (int i=1; i<books.size(); i++) {
            int currLevelHeight = books[i][1];

            miniHeight[i] = miniHeight[i-1]+currLevelHeight;
            int occupyWidth = books[i][0];

            for (int x=i-1; x >= 0; x--) {
                occupyWidth += books[x][0];
                currLevelHeight = max(currLevelHeight, books[x][1]);
                if (occupyWidth > shelfWidth) {
                    break;
                }
                int preLevelTotalHeight = 0;
                if (x-1 >= 0) {
                    preLevelTotalHeight = miniHeight[x-1];
                }
                miniHeight[i] = min(miniHeight[i], 
                    preLevelTotalHeight+currLevelHeight);
            }
        }

        return miniHeight.back();
    }
};
