# 29. 两数相除
* 难度：2
* 给定一个被除数和一个除数，求商。不能使用乘法，除法，余数运算。求得的商向 0 截断。注意题目要求返回值必须在 32 位整数所能表示的范围内。
* 解析：使用二进制的思想，将除数每次倍增2，商也会倍增。

174. 地下城游戏

  • 难度：2
  • 题意：给定一个方格代表一张地下城的地图，你需要从左上角出发到达右下角拯救公主。每个格子代表你会获得的血量，负值代表会掉血。当血量小于等于0 时立即死亡，任务失败。求初始最少需要多少血量。每次你只能向右或者向下移动一格。
  • 解析：动态规划从右下角开始推算最小需要的血量即可。
